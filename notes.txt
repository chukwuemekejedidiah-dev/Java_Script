*1. What is an object in JavaScript?*
An object is a collection of key-value pairs, where each key is a string (or Symbol) and the value can be any data type, including numbers, strings, booleans, arrays, functions, or even other objects.

*2. How is it different from variables like numbers or strings?*
Objects are complex data types that can store multiple values and functions, whereas primitive data types like numbers and strings store single values.

*3. Why do we use objects?*
Objects help organize related data and behavior, making code more structured and reusable.

*1. What is a property?*
A property is a key-value pair in an object, where the key is a string (or Symbol) and the value can be any data type.

*2. What is a method?*
A method is a function that is a property of an object.

*3. What is the difference between them?*
Properties typically store data, while methods perform actions or calculations.

*4. Difference between a function and a method*
A function is a standalone block of code, while a method is a function that belongs to an object.

## BASIC OBJECT CREATION
let student = {
  name: "John Doe",
  age: 20,
  course: "Computer Science"
};

console.log(student.name);
console.log(student["age"]);
console.log(student);
## METHODS (FUNCTIONS INSIDE OBJECTS)
student.introduce = function() {
  console.log(`Hi, my name is ${this.name} and I study ${this.course}`);
};
student.introduce();
## NESTED OBJECTS
student.address = {
  city: "Lagos",
  country: "Nigeria"
};
console.log(student);
